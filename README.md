Welcome to the code repository for the bachelor thesis Iterative Methods for Low-Rank Canonical Polyadic Tensor Approximation. 

"tensor generation.jl" handles the generation of synthetic tensors with given dimensions, rank, order and weight distribution with a given random generator seed.

"tensor operations.jl" contains functions performing tensor operations

"cp-als.jl" contains functions performing CP Alternating Least Squares 

"gradient descent.jl" contains functions computing CP objective function, gradient, and the kernel for gradient descent
